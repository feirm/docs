# Feirm Protocol Documentation
`docs` is the source for the Feirm protocol documentation built using [MkDocs](https://www.mkdocs.org/) along with the [MkDocs Material Theme](https://squidfunk.github.io/mkdocs-material/).

## Contributing
If you wish to contribute to the documentation, we suggest you join our [Discord](https://discord.gg/mUZdSKg) to bring yourself up to date with the latest activity about the project.

GitHub issues will be used to manage the requests for changes.

In order to submit any changes, please fork this repository and create a branch to put your changes in. Once you are happy with your changes, please submit a Pull Request back to the source repository for the changes to be reviewed.
